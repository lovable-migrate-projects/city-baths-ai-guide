## Цель итерации

Современный визуальный редизайн **главной страницы** портала "Городские бани": каталог заведений с интерактивной картой и ИИ-чатом для поиска. Только UI с моковыми данными — интеграцию с GraphQL и реальным ИИ-агентом вы подключите сами.

## Стек (минимальный, без лишнего)

- **TanStack Start** (React 19 + Vite, SSR из коробки) — базовый каркас Lovable.
- **styled-components v6** — единственный инструмент стилизации. Никакого Tailwind, shadcn, MUI.
- **Leaflet 1.9.4 + react-leaflet 4.x** — карты на OpenStreetMap.
- **react-markdown 10** — рендер ответов ИИ.
- **lucide-react** — иконки (поиск, звезда, часы, маркер, отправка). Если предпочитаете другую — скажите.
- **Google Fonts**: Manrope (заголовки) + Inter (текст), подключаем `<link>` в `__root.tsx`.

Готовые Tailwind/shadcn-файлы из стартового шаблона остаются нетронутыми и не используются — мы пишем собственные styled-components с нуля. Глобальные сбросы — через `createGlobalStyle`.

## Компоненты (все на styled-components)

1. **AIChatPanel** — карточка в hero. Поле ввода, кнопка отправки, чипы-подсказки, лента сообщений, markdown-рендер ответов, индикатор «печатает».
2. **AIChatFloating** — кнопка-пузырь + выезжающая панель/bottom-sheet с тем же AIChatPanel.
3. **CompanyCard** — фото, название-ссылка, адрес, рейтинг (звёзды), статус «Открыто/Закрыто сейчас», теги.
4. **CompaniesMap** — Leaflet+OSM, кастомные маркеры в фирменном синем, поповер с мини-карточкой при клике, синхронизация подсветки с карточкой в списке при наведении.
5. **ViewToggle** — Список / Карта (мобильный) и сортировка (десктоп).
6. **OwnerCTA** — секция «Это ваше заведение?» с кнопками «Заявить права» и «Разместить новое».
7. **Header / Footer** — заготовка для всего сайта.
8. **GlobalStyle + theme** — `ThemeProvider` styled-components с цветами/радиусами/тенями/брейкпоинтами.

## SSR и styled-components

styled-components корректно работает с SSR через свой `ServerStyleSheet`. Подключим babel/swc-плагин и серверную сборку стилей в шелл-компоненте, чтобы не было FOUC.

Leaflet рендерим только на клиенте: компонент карты грузим динамически и рендерим внутри гварда «есть ли window», CSS Leaflet импортируем внутри клиентского чанка.

## Данные (моки)

`src/data/companies.ts` — 12 заведений по Москве/СПб/Екатеринбургу с фото-плейсхолдерами и координатами. Типы `Company`, `ChatMessage` — публичный контракт, чтобы вам было легко заменить источник на ваш GraphQL API.

## Контракт ИИ-чата для вашей интеграции

```ts
type ChatMessage = {
  id: string;
  role: "user" | "assistant";
  content: string; // markdown
  companies?: Company[]; // если ответ содержит карточки
};

type AIChatProps = {
  messages: ChatMessage[];
  isStreaming?: boolean;
  onSend: (text: string) => void;
  suggestions?: string[];
};
```

Внутри компонента — заглушка с демо-ответом для превью; вы подмените `onSend` на свой агент.
